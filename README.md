## 📝 기획 의도

> 1. 업무 상, 그리고 그 외 목적으로 사용 중인 ***여러 캘린더***의 일정을 하나로 ***통합 관리*** ( Kakao, Google, Outlook)
> 2. 통합된 나의 일정을 내 그룹 캘린더의 ***멤버들과 공유***하여 모임 일정 결정에 도움
> 3. 그룹 일정에 사진을 포함한 ***피드와 댓글***을 통해 추억을 기록하고, 공유하는 커뮤니티 제공
